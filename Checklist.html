Lab 3 — Quiz Configurator

Sammanfattning och steg-för-steg checklista för att bygga WPF MVVM-applikationen (Configuration + Play, samt frivillig import via Open Trivia DB).

Snabb sammanfattning

Appen har två huvudlägen: Configuration (skapa/ändra question packs) och Play (kör quiz). För godkänt krävs MVVM, data binding, asynkron json-lagring, navigation mellan vyer, slumpning av frågor/svar, timer och resultat. För VG krävs import från Open Trivia DB via asynkront API-anrop och robust felhantering.

Projektstruktur (sätt upp tidigt)

Steg 1 — XAML & Layout (ingen C#-funktionalitet än)

Steg 2 — MVVM-bindning och kommandon

Steg 3 — Funktionalitet (G — krav)

Steg 4 — För väl godkänt (VG) — Import från Open Trivia DB

Steg 5 — Testning, UX & polish

Steg 6 — GitHub & inlämning

Tips Gör små iterativa commits, håll ViewModels enkla, mocka IO vid testning, och refaktorera tidigt om något blir rörigt.
Genvägar exempel Ctrl+N Ctrl+S Del
